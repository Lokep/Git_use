 # 当然还是记录一下，从本地上传至 ` gayhub ` 的整个过程

 * 第一步的话，应该就是在 ` gayhub ` 上新建一个 ` repository `

 * 第二步的话，在本地通过 ` git clone ` 将文件下载至本地

 * 然后 通过 ` git init ` 初始化

 * 然后使用 ` vi ` + ` file ` 的方式 新建文件，并进行编辑

 * 通过这种方式编辑的文件的话，第一次不知道怎么保存，也是靠猜的方式，
   
   按住 ` ESC `，然后 ` shift +  ;`,
 
 * 保存完了之后，就是上传了，上传的时候，要记得写commit，
   
   也就是 ` git commit -m 'description' ` ，

 * 通过 ` git push ` 提交至之前clone的那个仓库中
